Página Web para Blogs

Página web en la que los usuarios pueden escribir artículos o simplemente leerlos.
Desarrollado en PHP, gestor de base de datos MySql.

Características principales:

  • Registro de Usuarios
  • Inicio de Sesión
  • Página con las entradas al blog más recientes
  • Despliegue de entradas por categoría
  • Creación de categorías y entradas
  • Edición y eliminación de entradas
  • Página Web Responsive
